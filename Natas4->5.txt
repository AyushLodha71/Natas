Referer is the term used to find out the URL you are coming from.
So, for eg. on Instagram, you see a nice bag and now you want to buy it. So you click on the link of the bag. Suppose that the company who is selling the bag has requested referer, then your browser would give the URL of the Instagram page you used to come on that website.
Referer is requested in the Header.

So, in this question, install any header editor of your choice. I tried Mod Editor and Header Editor Lite. Both of them worked. Then add a tag with:
Header name: Referer
Header value: http://natas5.natas.labs.overthewire.org/

Then go on the website and reload. You will get the password. Now, If you go to Inspect->Network and select natas4.natas.labs.overthewire.org file and scroll down, in the ‘Request Headers’ section, you will see Referer which is now the url we set to. Now, if you remove it / change, it will again show the error message.
